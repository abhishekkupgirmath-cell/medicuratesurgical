Advanced Features
02

Multi-User Collaboration

The Multi-User collaboration function allows you to connect up to 100 people in a single VR room to present a lecture. Each connected student and teacher has their own unique 3D Avatar, which they can individually personalise. Connected students can be seen and heard as if together in the same room – wherever in the world they are currently in.
03

DICOM Viewer

Human Anatomy VR’s innovative algorithms can reconstruct 2D CT and MRI scan series, and produce a 3D model that you can see in VR as if the patient were right in front of you. The advanced Cutting feature allows you to cut the model in real time in any plane to access all internal structures. The Windowing function allows you to better visualize specific tissue types.
04

Web Manager

Test your students directly and easily in VR using the Web Test Manager. The automatic assessment of students’ test success saves valuable time. Results include statistical data, which enable you to improve the efficiency of your teaching.
